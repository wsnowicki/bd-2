# bd-2
Projekt z baz danych


## Uruchamianie
1. Utwórz bazę danych oraz użytkownika w postgresie:

Na systemie Linux:
```bash
python3 db_generator.py
```
lub na systemie Windows:
```bat
py db_generator.py
```

2. Utwórz plik .env i wypełnij go odpowiednimi ustawieniami dla twoich potrzeb.
```bash
cp .env.example .env
vim .env
```
